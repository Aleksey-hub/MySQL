1. Создать структуру БД Вконтакте по скриптам, приложеным в файле примеров examples.sql
	Подключаемся к ubuntu:
		C:\Users\Алексей>ssh aleksei@192.168.1.108
	Скачиваем файл examples.sql в директорию ~/lesson03 и редактируем (оставляем только строки относящиеся к БД vk):
		aleksei@ubuntu-MySQL:~/lesson03$ vim examples.sql
		aleksei@ubuntu-MySQL:~/lesson03$ mysql
		mysql> SOURCE examples.sql

		mysql> SHOW DATABASES;
		+--------------------+
		| Database           |
		+--------------------+
		| information_schema |
		| mysql              |
		| performance_schema |
		| sys                |
		| vk                 |
		+--------------------+
		5 rows in set (0,00 sec)


2. Используя сервис http://filldb.info или другой по вашему желанию, сгенерировать тестовые данные для всех таблиц, 
учитывая логику связей. Для всех таблиц, где это имеет смысл, создать не менее 100 строк. Загрузить тестовые данные.
	После генерации на сайте получаем файл "fulldb30-03-2020 16-58.sql". Для удобства переименовываем в fulldb.sql.
		mysql> USE vk;
		SOURCE fulldb.sql
